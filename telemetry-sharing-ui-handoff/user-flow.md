## Error States for Landing Page

| Condition                     | Suggested UI                                                                              |
| ----------------------------- | ----------------------------------------------------------------------------------------- |
| No `#` fragment in URL        | Show default landing page / CTA to use the plugin                                         |
| Decode fails (invalid base64) | "This link appears to be invalid or corrupted. Try generating a new one from the plugin." |
| Gunzip fails                  | Same as above                                                                             |
| JSON parse fails              | Same as above                                                                             |
| Unknown `schema_version`      | "This link was generated by a newer version of the plugin. Please update."                |
| All fields present            | Normal partner matching view                                                              |
